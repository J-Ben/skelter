# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, open a private security advisory via the **Security** tab on GitHub, or contact the maintainer directly.

You can expect a response within 72 hours. If the issue is confirmed, a patch will be released as soon as possible.
