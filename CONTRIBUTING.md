# Contributing

Thanks for your interest in contributing to react-zero-skeleton.

## Setup

```bash
npm install
npm run build
npm test
npm run typecheck
```

## Workflow

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Add a changeset: `npx changeset`
4. Open a pull request against `main`

## Reporting bugs

Open an issue with a minimal reproduction. Include your React / React Native version and the platform (web, iOS, Android).

## Proposing features

Open an issue first to discuss the idea before writing code.

## Code style

TypeScript strict mode. No `any`. Run `npm run typecheck` before submitting.
